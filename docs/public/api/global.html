Gravitational acceleration on the Z-axis (usually 0 if Z is depth not height). If Z was your vertical jump axis, this would be non-zero and GRAVITY_Y might be 0. For our X (horizontal), Y (vertical screen, jump/fall), Z (depth) convention, Z gravity is 0.
